## Personalisasi NIM

* NIM saya: `244107060087`.
* 3 digit terakhir NIM = `087` dikonversi menjadi `87.0`.
* Nilai ditambahkan dalam list harga barang sebagai elemen ke-6.

Setiap mahasiswa akan memiliki total belanja sesuai NIM masing-masing. Setelah total dihitung, sistem akan menentukan diskon berdasarkan aturan.
* 10% jika > 200.000.
* 5% jika 100.000 - 200.000.
* Selain itu, tidak ada diskon.
